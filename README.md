# MIEMards

## Scripts

-   `npm i` - подтянуть все зависимости
-   `npm run dev` - запустить дев версию на локальном сервере
-   `npm run build` - сбилдить проект
-   `npm run preview` - запустить сбидженое на локальном сервере
-   `npm run lint` - прогнать линтер

## Stack

-   Vite
-   React
-   TypeScript
